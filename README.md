A URL shortener REST API built with Node.js, Express, and MongoDB.

## Live Demo
Frontend: https://linksnip-frontend.onrender.com

## Tech Stack
- Node.js and Express
- MongoDB and Mongoose
- JWT Authentication
- QR Code Generation

## Features
- User registration and login
- Shorten long URLs
- Custom aliases
- QR code generation
- Click tracking
- Link expiry
